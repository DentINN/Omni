# Omni

**One inbox. Every conversation.**

Omni is a next-generation communication hub that unifies multiple messaging platforms into a single, intelligent interface.

## Vision

People shouldn't have to think about where a conversation happened.

Whether a message comes from Telegram, WhatsApp, MAX or another platform, communication should feel seamless.

Omni is built around people, not messengers.

## Mission

Create the most intuitive communication experience by combining messaging, intelligent organization, privacy, and AI assistance into one application.
